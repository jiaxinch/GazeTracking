Class VirtualInteractorAgent<TInteractor, TData>

Basic implementation of an UI framework agnostic IInteractorAgent. This generic base class is meant to be inherited and lacks implementations of AddInteractorFor and AddInteractorsFor to complete the common (implicit) ProviderInteractorAgent usage contract.
