Class Host

The main class for communicating with EyeX Engine. Hosts one EyeX Context. It is possible to create more than one host in a process.

System.String processId

The process this agent will handle queries for. If not specified or null, the agent will be registered to handle queries for the current process. A process id is usually an integer, but it should be converted to a string for usage here. May be empty or null, and is then treated as the current process.
