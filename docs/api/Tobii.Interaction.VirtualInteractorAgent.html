Class VirtualInteractorAgent

Default implementation of an UI framework agnostic IInteractorAgent using VirtualInteractor as interactor type. Everything is taken care of. VirtualInteractor are moved by using the Move(VirtualInteractor, Vector2), MoveTo(VirtualInteractor, Vector2) and MoveCenterTo(VirtualInteractor, Vector2) extension methods. extension methods or by explicitly setting the CachedBounds property.
