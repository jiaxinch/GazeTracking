    Introduction

    The following articles will guide you through the main tools and concepts provided by the Tobii Core SDK, based on the functionality powered by Tobii Eye Tracking Core Software.

    In this introduction, you will find a brief overview of the articles. Various code samples are shown as a complement to the detailed API Documentation.

    Articles

    • Introduction, provides a short overview of the topics in this guide.

    • Getting started, shows how to setup the environment to run the code samples in an ordinary Console- or Windows Presentation Foundation (WPF) application.

    • Interactors, describes how to define regions in your application's window which can react on gaze, followed by several possible behaviors.

    • Virtual Windows, introduces the concept of virtual windows, which helps you define interactors outside of the application's window or attach them to some other application's windows.

    • WPF, shows how to add gaze based features to a WPF application.

    • Streams, provides the basic concepts of retrieving and processing raw data from the eye tracker.

    • States, provides an overview of global states defined in Tobii Eye Tracking Core Software, e.g. user presence.
