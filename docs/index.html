Welcome to Tobii Core SDK!

    The Tobii Core SDK provides you with an API and framework to build applications enhanced with the knowledge of the user's gaze and attention - more intuitive natural user interfaces, deeper immersion, next level experiences, or something never seen before!

    This introductory documentation will show you how easy it is to get started using eye tracking from Tobii to create these powerful experiences using e.g. the knowledge of where the user looks at the screen or what part of your user interface that holds his or her attention. In addition to gaze, we also provide information about the users head position, user presence and more.

    If you prefer to dive right in to the code, check out our samples.

    In order to get started, make sure you have your eye tracker up and running. When everything is working you should see the red lights glowing slighty from the eye tracker underneath your screen and two white dots in the Tobii tray application indicating your eye position in the eye trackers field of view. Run "Tobii Eye Tracking" to start the Tobii software if you have quit it before.

    If you do not have Tobii software yet, please head to our website and download the Tobii Eye Tracking Core Software.

    If you have developer questions we have a forum where you can ask them.

    If you experience any sort of issues, our support is here to help you.
